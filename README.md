# Mesto

Проект Яндекс Практикума **"mesto"** был сверстан при прохождении 3 семестра на курсе веб разработки.

## Функциональность.
> *На сайте представлены места для отдыха по России*

## Технологии
> *При верстке сайта были использованы следующие технологии*
* Флексбокс
* Grid layout
* Позиционирование
* JavaScript

## Планы по доработке
